# todo-list
A simple and functional To-Do List web application that allows users to manage daily tasks with priorities, categories, and due dates. The system is built using PHP, MySQL, and phpMyAdmin, making it easy to run on any local server such as XAMPP.

## Features
• Add Tasks with name, category, priority, and due date
• Categorized Task View (home, education, health, family, etc.)
• Priority Levels: low • medium • high
• Due Date Tracking
• Update Status: mark tasks as pending or complete
• Delete Tasks anytime
• Persistent Storage using MySQL
• Simple & Clean UI (Achuchu-style neatness)

## Tech Stack
• Frontend - HTML, CSS
• Backend - PHP  
• Database - MySQL / MariaDB
• Environment - XAMPP, phpMyAdmin
